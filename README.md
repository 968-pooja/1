# Linux Server Setup Project

## Objective
Configured Ubuntu Linux server on VirtualBox and deployed a web server.

## Technologies Used
- Ubuntu Server
- Nginx
- VirtualBox

## Tasks Performed
- Installed Ubuntu on VirtualBox
- Created users and groups
- Configured file permissions (chmod, chown)
- Installed and configured Nginx
- Hosted a static HTML page
- Verified web server using localhost

## Commands Used
- apt
- systemctl
- chmod
- chown
- nano
- ip a
